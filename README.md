# README

* 软件功能说明

  本程序完成了一个简易的交互的绘图系统，实现了基础的图形学算法，包括线画图元生成、贝塞尔曲线生成、B样条曲线生成、二维图形的裁剪和变换。同时可以满足画布的重置、保存、设置画笔等交互功能。

* 开发环境说明
  * MacOS 10.15.6 
  *  PyCharm 2020.3.3。
    * Python 3.7.7
      * numpy 1.23.0、Pillow 9.2.0、PyQt5 5.15.7
